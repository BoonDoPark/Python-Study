# Calculator
## Overview
해당 프로젝트는 **계산기를 구현**을 목표로 삼고 있습니다.
계산기란, 각종의 수 계산을 신속히 실용적으로 할 수 있는 기구나 기계의 총칭
즉, 기본적인 **사칙연산으로 계산**을 할 수 있습니다.
